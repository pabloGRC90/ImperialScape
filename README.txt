Tras hacer Login el usuario puede empezar partida, esta empezará con un número aleatorio de 3 a 5 jugadores.

Al inicio el enigma se designará a un jugador de forma aleatoria.
Tras resolverse el primer enigma será el almirante quien decida quier resolverá cada uno de los siguientes

Si se acierta se añade una llave.
Si se falla se restan 60 segundos (no implementado el control del almirante para las penalizaciones por fallo)

Si se consiguen las 5 llaves antes de terminar el tiempo (se descuenta cada error), se consigue salir con éxito.
